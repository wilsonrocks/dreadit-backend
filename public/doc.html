NorthCoders News API

All endpoints return an object with a single key with the information nested as the value.

/api/users

Fields returned for user objects: '_id', 'name', 'username', 'avatar_url'.

GET /api/users/:_id

Returns information about the user with the given _id.

GET /api/users

Returns information about all users.

GET /api/users/:_id/articles

Returns all the articles by a given user.

/api/topics

Fields returned for this endpoint: '_id', 'title', 'slug'.

GET /api/topics

Returns an array of all topics.

GET /api/topics/:_id/articles

Returns an array of all articles for the given topic.

POST /api/topics/:_id/articles

Creates a new article for the topic with the given _id. The body should contain a JSON object with title and body keys.

/api/articles

Fields returned by this endpoint: '_id', 'title', 'body', 'belongs_to', 'votes', 'created_by'.

GET /api/articles

Returns an array of all articles.

GET /api/articles/:_id

Returns a single article.

PUT /api/articles/:_id

Votes the article given by _id up or down. The query string must contain a parameter 'vote' with value 'up' or 'down'.

GET /api/articles/:_id/comments

Returns all comments for the article given by _id.

POST /api/articles/:_id/comments

Creates a new comment. The body must be a JSON object with the key comment and the text of the comment as the value.

/api/comments

Fields returned by this endpoint: 'created_at', '_id', 'body', 'belongs_to', 'created_by', 'votes'.

PUT /api/comments/:_id

Votes the comment given by _id up or down. The query string must contain a parameter 'vote' with value 'up' or 'down'.

DELETE /api/comments/:_id

Deletes the comment given by _id and returns it.
